# 制作背景

サービスの概要は、学習を効果的に管理し、直近の学習の進捗を追跡するための便利な学習記録アプリです。想定しているユーザーは高校、大学の受験生や学習する習慣がある人を想定しています。目標設定や学習記録を可視化することでユーザーのモチベーションを高め、学習の継続を支援するという課題を解決することを目指して作りました。
